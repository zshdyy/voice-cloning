## 🎚️ 参数速查表

### 最常用的参数组合

```python
# 轻微女化
convert_gender_high_quality(input, output, pitch_ratio=1.65, formant_ratio=1.12)

# 自然女声（推荐首选）⭐
convert_gender_high_quality(input, output, pitch_ratio=1.75, formant_ratio=1.15)
# 或使用默认值：
convert_gender_high_quality(input, output)

# 高亢少女
convert_gender_high_quality(input, output, pitch_ratio=1.90, formant_ratio=1.18)

# 女声变男声
convert_gender_high_quality(input, output, pitch_ratio=0.65, formant_ratio=0.85)
```

### 参数含义

| 参数 | 范围 | 说明 |
|------|------|------|
| `pitch_ratio` | 1.6-2.0 | 越大音调越高；1.75 为推荐值 |
| `formant_ratio` | 1.1-1.2 | 越大音色越女性化；1.15 为推荐值 |

## ❓ 常见问题速答

**Q: 为什么转换很慢？**  
A: WORLD 声码器需要逐步进行特征提取和合成，处理时间通常是音频时长的 10-30 倍。这是保证高质量的必要成本。

**Q: 转换质量不理想怎么办？**  
A: 调参！先用默认参数 (1.75, 1.15) 测试，然后微调找到"甜区"。

**Q: 支持哪些音频格式？**  
A: WAV, MP3, FLAC, OGG 等（librosa 自动支持）。

**Q: 能否女声变男声？**  
A: 可以！使用反向参数：pitch_ratio=0.65, formant_ratio=0.85

**Q: 如何处理大量音频文件？**  
A: 使用 `batch_convert()` 函数进行批量处理。
